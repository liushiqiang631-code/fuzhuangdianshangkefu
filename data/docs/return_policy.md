
# 退换货政策

## 一、退换货条件
1. 自签收之日起7天内可申请退换货
2. 商品须保持原样：未穿着、未洗涤、未修改，吊牌和包装完好
3. 赠品需一并退回
4. 定制商品、贴身衣物（内衣、泳装）不支持无理由退换

## 二、退换货流程
1. 在"我的订单"中找到对应订单，点击"申请退换"
2. 选择退换原因，上传商品照片
3. 等待审核（1个工作日内）
4. 审核通过后，按指引寄回商品
5. 收到商品并验收合格后，1-3个工作日内退款

## 三、运费说明
- 质量问题：运费由商家承担，提供运费补偿
- 非质量问题：运费由买家承担
- 包邮商品退货需扣除原订单运费

## 四、退款方式
- 原路退回：退回至原支付账户
- 退款时效：验收后1-3个工作日
- 银行卡退款可能需额外3-5个工作日到账

## 五、特殊说明
- 大促期间（双11、618等）退换货处理时间可能延长至5个工作日
- 如遇争议，以商品实物照片和质检报告为准
- 恶意退换货（频繁退货、调包等）将被限制退换货权限
